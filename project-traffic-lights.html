Traffic Lights

Technology Stack

SystemVerilog


Objective

This two-part project is an implementation of two versions of an intersection. Part 1 is an implementation of a symmetrical intersection in which our model will have 4 possible states.



The four states that we have are as follows:


  • North/Sound bound = GREEN, East/West bound = RED
  • North/Sound bound = YELLOW, East/West bound = RED
  • North/Sound bound = RED, East/West bound = GREEN
  • North/Sound bound = RED, East/West bound = YELLOW
